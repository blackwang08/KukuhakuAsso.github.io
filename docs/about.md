# 关于我们

空白解谜组成立于2026年5月，是一支以各类解谜活动（如WIG、ARG等）为核心的玩家与创作团队。成员来自多个不同领域，致力于将多元化解谜视角带给大众。同时，空白组持续探索解谜设计的新边界，尝试打造多样化、风格化的谜题体验。团队名称取自《游戏人生》主角“空”和“白”，口号为“空白永不败北”。
